# Cursor rules (orpheus_client) — авто-коммит

## По умолчанию
Если пользователь **не запретил коммит**, после внесения изменений:
- обнови артефакты (CHANGELOG/AI_WORKLOG/docs при необходимости)
- прогоняй проверки по необходимости
- создай коммит через:

```powershell
.\scripts\auto-commit.ps1
```

## Если пользователь запретил коммит
- Не делай `git commit`.
- Всё равно обновляй артефакты процесса (если были изменения кода/конфигов).

## Сообщения коммитов
- Формат: `type(client): описание`
- Типы: `feat`, `fix`, `docs`, `chore`, `refactor`, `test`
