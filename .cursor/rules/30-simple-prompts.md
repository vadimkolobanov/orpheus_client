# Простые промпты (шпаргалка)

Если пользователь пишет коротко — не спрашивай уточнения, а действуй:

## Ключевые слова → Действия:

| Промпт | Что делать |
|--------|-----------|
| "баг", "не работает", "сломалось" | Найти и исправить проблему |
| "пуш", "push", "запуш" | git add + commit + push |
| "коммит", "commit" | git add + commit |
| "статус", "status" | git status всех репозиториев |
| "тест", "test" | flutter test |
| "билд", "build" | flutter build apk --release |
| "звонок", "call" | Работа с call_screen.dart и WebRTC |
| "чат", "chat" | Работа с chat_screen.dart |
| "уведомления", "notifications" | notification_service.dart |

## Правила:

1. **Не спрашивай очевидное** — если написано "запуш сервер", пуш сервер
2. **Действуй сразу** — изучил код → исправил → закоммитил
3. **Отчёт кратко** — таблица или список, без воды
4. **Помни контекст** — если работали со звонками, значит продолжаем со звонками
