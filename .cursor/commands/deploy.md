# Command: deploy

Деплой сервера или сборка приложения.

## Спроси:
- **Что деплоить?** (server / client / both)

## Для сервера (Python backend):
1. Закоммить и запушить `D:\Programs\orpheus`
2. Сообщить что нужно задеплоить на хостинге

## Для клиента (Flutter):
1. `flutter build apk --release`
2. APK будет в `build/app/outputs/flutter-apk/`

## Для обоих:
Сделай оба пункта последовательно.
