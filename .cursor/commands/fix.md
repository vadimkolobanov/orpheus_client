# Command: fix

Пользователь описал баг или проблему.

## Твои действия:
1. **Найди причину** — изучи релевантный код
2. **Исправь** — минимальные изменения
3. **Проверь** — `flutter test` если нужно
4. **Закоммить** — через `.\scripts\auto-commit.ps1`
5. **Отчёт** — кратко что было/стало

Не спрашивай "какой файл?" — найди сам по описанию проблемы.
