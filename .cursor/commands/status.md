# Command: status

Покажи текущий статус проекта.

## Проверь:
1. **Git статус** всех репозиториев (client, server, desktop, site)
2. **Незакоммиченные изменения**
3. **TODO/FIXME** в коде (если есть)
4. **Последние коммиты** (1-2 на репу)

## Формат ответа:
Кратко, таблицей или списком. Без лишних слов.
