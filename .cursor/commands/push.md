# Command: push

Запушить изменения в git.

## Что делать:
1. Проверь `git status` во всех нужных репозиториях
2. Если есть незакоммиченные изменения — закоммить
3. `git push origin <branch>`
4. Сообщи результат

## Репозитории проекта:
- `D:\orpheus_client` — Flutter клиент
- `D:\Programs\orpheus` — Backend Python
- `D:\orpheus_desctop` — Desktop клиент
- `D:\Orpheus_Site` — Сайт
