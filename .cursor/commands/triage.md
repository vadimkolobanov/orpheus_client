# Command: triage (orpheus_client)

Цель: быстро локализовать проблему (тесты/сборка/рантайм).

## Минимальный сценарий
1. Сними симптомы:
   - точная ошибка/стек
   - какой экран/сервис
   - условия воспроизведения
2. Быстрые проверки:
   - `flutter test` (или конкретный тест)
   - `flutter analyze` (если есть варнинги/типы)
3. Локализация:
   - `git diff` → что меняли рядом с падением
   - найти точку входа (UI → handler → service → model)
4. Фикс:
   - минимальный, с тестом на регрессию (если уместно)
5. Артефакты:
   - `CHANGELOG.md` / `AI_WORKLOG.md`
