# Orpheus Client — установка и запуск

## Требования
- Flutter SDK (версия задаётся в `pubspec.yaml` → `environment`)
- Android SDK / Android Studio (для запуска на Android)

## Установка зависимостей

```powershell
flutter pub get
```

## Запуск

```powershell
flutter run
```

## Быстрые проверки

### Запуск тестов

```powershell
flutter test
```

### Быстрый прогон без отчётов

```powershell
.\quick_test.ps1
```

### Прогон с сохранением отчётов

```powershell
.\test_runner.ps1
```

Отчёты сохраняются в `test_reports/`.
