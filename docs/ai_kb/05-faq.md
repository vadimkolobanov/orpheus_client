# FAQ

## Почему нет логина по телефону/email?
Orpheus использует **идентичность по криптографическому ключу**. Это снижает зависимость от операторов/почты и уменьшает объём персональных данных в системе.

## Что является моим “ID”?
Обычно — **публичный ключ**. Им делятся для добавления контакта (часто через QR).

## Почему контакт/чат “не работает”, хотя я добавил(а) человека?
В текущей логике удобнее всего, когда добавление сделано **в обе стороны**:
- вы добавляете человека по его ID/QR;
- он добавляет вас по вашему ID/QR.

## Почему в уведомлениях нет текста сообщения?
Это сознательный компромисс ради приватности: push‑уведомления не должны раскрывать содержимое переписки.

## Почему я не могу сделать скриншот/запись экрана?
На Android приложение включает защиту от скриншотов/записи экрана (это сделано для приватности).
Для обмена данными используйте:
- **Профиль → «Поделиться контактом»**, или
- копирование ID (тап по строке ключа), или
- описание проблемы в чате поддержки + отправку логов (кнопка со скрепкой).

## Что такое Duress mode?
Это режим, в котором приложение выглядит “пустым” (минимум данных в UI). Он предназначен для ситуаций принуждения, но **не равен** гарантированному стиранию данных с устройства.

## Можно ли восстановить доступ, если я потерял ключи/устройство?
Если доступ к вашей идентичности завязан на приватный ключ, то при его потере восстановление может быть **невозможно**. Если в приложении есть функции резервного копирования/импорта — используйте их заранее.
