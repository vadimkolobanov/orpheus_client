# Что такое Orpheus

Orpheus — приложение для приватной коммуникации: **сообщения и звонки** с фокусом на privacy и устойчивость в “реальных” условиях (плохая сеть, фоновые режимы).

## Ключевые идеи
- **End‑to‑end шифрование сообщений (E2E)**: содержание сообщений шифруется на устройстве и не должно быть доступно серверу.
- **Идентичность по ключу**: в Orpheus нет логина по телефону/email — “ID” пользователя это **публичный ключ**.
- **Минимизация данных на сервере**: сервер нужен для доставки зашифрованных пакетов/сигналинга, а не как хранилище вашей переписки.

## Важно понимать (границы обещаний)
- **Анонимность ≠ неуязвимость**: если устройство скомпрометировано (root/malware/forensics), ни одно приложение не может гарантировать безопасность данных.
- **E2E ≠ “ничего не хранится на телефоне”**: история сообщений хранится локально ради удобства.
- **Duress mode** (режим принуждения) — это **скрытие данных в интерфейсе**, а не обязательное физическое уничтожение данных на диске.
