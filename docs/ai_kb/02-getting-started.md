# Быстрый старт для пользователя

## Установка
1. Скачайте APK с официальной страницы проекта: [orpheus.click](https://orpheus.click).
2. Если Android попросит — разрешите установку из “неизвестных источников” для вашего браузера/файлового менеджера.
3. Установите приложение и откройте его.

## Навигация в приложении (где что находится)
Внизу четыре вкладки:
- **Система** — статус сети/режима и подсказки, если связь нестабильна.
- **Контакты** — список контактов и личные чаты (E2E‑шифрованные).
- **Комнаты** — групповые чаты и доступ к **Сейфу** (личные заметки).
- **Профиль** — ваш ID/QR, безопасность, поддержка, обновления.

## Первый запуск
- При первом запуске приложение предложит выбрать **язык** (русский, английский или системный).
- Далее приложение создаёт **ключевую пару** (или предлагает создать/импортировать).
- Ваш "ID" — **публичный ключ**. Он нужен для добавления контактов.

## Добавление контакта и чат
Обычно контакт добавляется одним из способов:
- **QR/сканирование** (быстрее и меньше ошибок), или
- **ввод/вставка публичного ключа** вручную.

Важно: добавление контакта работает корректнее, когда сделано **в обе стороны** — вы добавили человека по его ID/QR, и он добавил вас по вашему ID/QR.

После добавления контакта можно:
- отправлять сообщения;
- получать сообщения и уведомления (без показа текста сообщения в пуше).

### Переименование контакта
Переименование **локальное** (видно только вам):
**Контакты → долгое нажатие на контакт → Переименовать контакт**.

## Сейф заметок
**Где**: вкладка **Комнаты**, самый верх списка — карточка **«Сейф»**.

**Как добавить заметку вручную**:
1. Откройте «Сейф».
2. Введите текст в поле ввода.
3. Нажмите **«Сохранить»**.

**Как сохранить сообщение из чата**:
1. **Долгое нажатие** на сообщение (в личных чатах, комнатах или у Оракула).
2. Выберите **«Добавить в заметки»**.

Заметки из чатов помечаются источником (контакт/комната/Оракул).

## Звонки
- Звонки устанавливаются через сигналинг, а медиа‑поток идёт напрямую между устройствами (P2P), когда это возможно.
- Качество звонка зависит от сети. В некоторых сетях может потребоваться дополнительная “помощь” инфраструктуры (это нормально для WebRTC‑приложений).

## Как поделиться своим ID
Откройте вкладку **Профиль**:
- можно показать **QR**;
- можно нажать **«Поделиться контактом»** (приложение сформирует текст с вашим ключом);
- можно тапнуть по строке с ключом, чтобы **скопировать ID**.

Примечание: в целях безопасности на Android в приложении включена защита от скриншотов/записи экрана, поэтому для обмена используйте **поделиться/копировать**, а не скриншоты.

## Рекомендуемые настройки безопасности
- **PIN‑код**: включите, если используете телефон в риск‑сценариях.
- **Duress mode**: используйте как “правдоподобное отрицание” — он скрывает данные в UI, но не является гарантированным стиранием.
- **Wipe/экстренное удаление**: включайте и используйте только если понимаете последствия (после удаления данные могут быть недоступны).
