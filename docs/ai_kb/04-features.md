# Возможности

## Личные сообщения
- Переписка с **E2E‑шифрованием**.
- История сообщений хранится локально на устройстве.

## Комнаты (групповые чаты)
- **Комнаты** — это групповые чаты для общения нескольких участников.
- Создание комнаты: вы получаете **пригласительный код** (invite code), которым можно поделиться с другими.
- Присоединение: по invite‑коду можно войти в комнату.
- Для владельца комнаты доступны:
  - **ротация invite‑кода** — старый код перестаёт работать;
  - **очистка истории** (panic clear) — удаляет сообщения у всех участников.
- Обычные участники могут **выйти из комнаты**.

**Важно**: сообщения в комнатах **не являются E2E‑шифрованными** — они хранятся на сервере для доставки всем участникам. Не передавайте чувствительные данные в комнатах.

## Сейф заметок
- **Сейф** — личные заметки, которые хранятся **локально** на устройстве.
- В «Сейф» можно:
  - писать свои заметки;
  - сохранять важные сообщения из чатов и комнат.
- Источник заметки отмечается:
  - из чата с контактом;
  - из комнаты;
  - из чата с Оракулом.

## Звонки
- Голосовые звонки через WebRTC (в идеале — P2P).
- Устойчивость к проблемам сети (переподключения, восстановление сессии при смене сети — когда возможно).

## Устойчивость “в реальности”
- Переподключение при потере сети.
- Очередь исходящих действий, когда связь нестабильна: приложение пытается “дослать”, когда соединение восстановится.
- Механизмы fallback по хостам/доменам (если основной адрес временно недоступен).

## Уведомления
- Push и/или локальные уведомления о событиях.
- По умолчанию уведомления стараются **не раскрывать** текст сообщения.
- В комнатах пользователь может отключить уведомления для конкретного чата.

## Режимы защиты
- PIN‑код для доступа к приложению.
- Duress mode (режим принуждения): скрывает реальную переписку/профиль в интерфейсе.
- Wipe (экстренное удаление данных): предназначен для ситуаций риска.
