# Возможности

## Сообщения
- Переписка с **E2E‑шифрованием**.
- История сообщений хранится локально на устройстве.
- **Комнаты (Rooms)** — групповые чаты. Создавайте комнаты, приглашайте участников по инвайт-коду, общайтесь группой. **Важно**: сообщения в комнатах хранятся на сервере (не E2E) — не передавайте чувствительные данные.
- **Хранилище заметок (Notes Vault)** — сохраняйте важные сообщения длинным нажатием. Работает для личных чатов, комнат и Oracle. Все заметки зашифрованы.

## Oracle of Orpheus
- **AI-помощник** встроен в приложение и всегда первый в списке контактов.
- Отвечает на вопросы о функциях приложения, помогает с настройками.
- Работает на русском и английском языках.
- Сохраняет контекст разговора для более полезных ответов.

## Звонки
- Голосовые звонки через WebRTC (в идеале — P2P).
- Устойчивость к проблемам сети (переподключения, восстановление сессии при смене сети — когда возможно).
- Работают даже в спящем режиме — входящие звонки проходят при блокировке экрана.

## Устойчивость "в реальности"
- Переподключение при потере сети.
- Очередь исходящих действий, когда связь нестабильна: приложение пытается "дослать", когда соединение восстановится.
- Механизмы fallback по хостам/доменам (если основной адрес временно недоступен).

## Уведомления
- Push и/или локальные уведомления о событиях.
- По умолчанию уведомления стараются **не раскрывать** текст сообщения.

## Интерфейс
- **Выбор языка** — переключайтесь между русским и английским в настройках.
- Современный дизайн с тёмной темой.
- Многострочный ввод сообщений.

## Режимы защиты
- **PIN‑код** для доступа к приложению — выбирайте длину: 4 цифры (быстрый вход) или 6 цифр (повышенная безопасность).
- **Duress mode** (режим принуждения): скрывает реальную переписку/профиль в интерфейсе.
- **Wipe** (экстренное удаление данных): предназначен для ситуаций риска.
- **Auto-lock** — автоматическая блокировка приложения после периода неактивности.
