# Orpheus Client — обзор проекта

## Назначение
`orpheus_client` — Flutter‑клиент для проекта Orpheus: защищённого мессенджера с end‑to‑end шифрованием и принципом минимизации данных на сервере.

## Область этой документации
Эта папка описывает **только Flutter‑клиент**:
- структура кода, ключевые модули/сервисы;
- правила разработки и тестирования;
- процесс релиза клиента и связь с публичными release notes.

Документация по серверу и админ‑панели хранится в соответствующих репозиториях/папках.

## Ключевые свойства клиента
- **Сообщения**: шифрование на клиенте, хранение в локальной БД.
- **Звонки**: сигналинг через WebSocket и аудио через WebRTC.
- **Безопасность приложения**: опциональный PIN, режим duress, механизмы wipe (см. ADR).
